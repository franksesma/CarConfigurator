# CarConfigurator
ASP.NET Web App Project. Car Configurator where user can customize a vehicle and order a car.

To run locally on your machine: 
1. Clone the repository from GitHub into Visual Studio.
2. Ensure "Allow invalid certificates for resources loaded from localhost" in chrome://flags/ (or make the similar changes in your preferred browser)
3. Ensure that SQL Server Data Tools are available with your Visual Studio. Modify your Visual Studio using the installer and find it within the individual components to install it
